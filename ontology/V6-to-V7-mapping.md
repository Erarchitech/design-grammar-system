# V6 -> V7 IRI Mapping (recovery-only)

**Purpose:** This file maps every renamed V6 ontology name to its V7 successor. It exists as a recovery guard for existing publications (documents, Cypher, external systems) that reference V6-era names -- it is NOT a runtime consumer artifact. Generated by `apply_v7_rename.py` from its own `RENAMES` list (the same list the rename transforms above consume), so this table can never drift from what was actually applied to produce `DesignGrammar-V7.owl`.

Authority: `ontology/V7-INVESTIGATION.md` (Phase 13 plan 13-01) locks every row below.
