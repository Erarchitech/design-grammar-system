### Анализ ризонера (по запросу)
- Живыми пробами доказано: сегодня **ни одно** правко-порождённое изменение OntoGraph не может уронить consistency-чек — все 65 subClassOf идут из статического TBox; живой проект добавляет только декларации. Посеянная как живые Neo4j-рёбра контрадикция → HermiT говорит «consistent» (export плоский). Это подтверждённое следствие ADR-820-1.
- Предложение (в `.planning/research/REASONER-VALUE-AND-AUTOREPAIR.md`): V1 event-driven consistency-gate после ingest/edit → V2 эмиссия таксономии (SUBCLASS_OF) на ingest → механизм починки со ступенчатой автономией L0 report / L1 propose-with-approval (diff-preview + apply + auto-recheck) / L2 opt-in auto-quarantine (обратимый флаг, без удаления). LLM только объясняет, никогда не применяет починку (эмпирика 2025: correctness слишком низкая).
- Perplexity MCP был недоступен — SOTA-ссылки собраны через web search (Horridge justifications, hitting-set repair, xpSHACL, LLM graph-repair empirical study).
