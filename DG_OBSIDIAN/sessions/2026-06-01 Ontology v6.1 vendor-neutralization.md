# Session: Ontology v6.1 — Vendor-Neutralization (Speckle/Rhino)

## Goal

Remove all technology-specific references from the DG ontology except the two permitted exceptions (Grasshopper, IFC). The ontology must model **common concepts** and remain neutral to any particular software stack. External standard-vocabulary alignments (W3C SWRL/PROV-O/SOSA/SHACL/SKOS/DCTerms/GeoSPARQL, BOT, Topologic) are legitimate links and were retained.

## What Was Done

### Scope
Three V6 ontology files. Topologic/BOT extension untouched (external-vocabulary links, not coupling).

### `ontology/DesignGrammar-V6.owl`

1. **Property renamed:** `dgv:speckleProjectId` → `dgv:externalProjectId` (label + comment neutralized).
2. **Comments scrubbed (Speckle/Rhino):**
   - Layer-3 header: "Speckle viewer" → "external 3D viewer"
   - `IntegrationConfig` class comment: dropped "Speckle integration / Speckle project ID"
   - `ValidationRun` class comment: "the Speckle version created" → "the external model version created"
   - `GeoRef` class comment: "(Speckle objectId, Rhino GUID, etc.)" → "(an external source-system object handle)"
   - `baseModelId`, `baseModelName`, `validationModelId`, `baseVersionId`, `validationVersionId` property comments: "in Speckle" → "in the external platform"
   - `provider` comment: "(e.g. 'Speckle')" → generic description
   - `geoRefId` comment: "(e.g. Speckle objectId, Rhino GUID)" → neutral
3. **ABox instances neutralized:**
   - `dgv:SpeckleIntegration_TestA` → `dgv:Integration_TestA` (label, comment, provider value, property use)
   - `dgv:GR_speckle_xyz1` → `dgv:GR_geom_1` (geoRefId `speckle:obj:xyz1` → `obj:xyz1`)
   - `dgv:GR_speckle_xyz2` → `dgv:GR_geom_2` (geoRefId `speckle:obj:xyz2` → `obj:xyz2`)
   - `dg:OS_b1_var_b`: updated `hasGeoRef` refs to renamed IRIs; `objectRefName` `rhino-guid-9f3e2a` → `obj-9f3e2a`
4. **Version bump:** 6.0 → 6.1, changelog note appended to `rdfs:seeAlso`.

### `ontology/DesignGrammar-standards-extension-V6.owl`

- `geo:Geometry` comment: "a handle into a source-system (Speckle/Rhino)" → "a handle into an external source-system".
