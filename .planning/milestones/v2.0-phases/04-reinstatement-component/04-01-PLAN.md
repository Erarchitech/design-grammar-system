Create `DG/src/DG.Core/Services/DesignStateReinstatementService.cs` — a stateless service that implements the pre-validation algorithm from D-05/D-06 without any Grasshopper dependencies.

Public API:
```csharp
public sealed class DesignStateReinstatementService
{
    public ReinstatementResult Validate(
        DesignStateSnapshot snapshot,
        IReadOnlyList<ResolvedTarget> resolvedTargets,
        string? lastAppliedStateId);
}
```

Algorithm (implements D-05 through D-13):

1. **StateId guard (D-09):** If `lastAppliedStateId` is not null and equals `snapshot.StateId`, return result with all parameters set to `Unchanged`, `Applied = false`, `Aborted = false`.

2. **Build parameter-to-target map:** For each parameter in `snapshot.Parameters`, find the matching `ResolvedTarget` by `ParameterId` (case-sensitive ordinal match). If no matching target found in the list, status = `MissingTarget` with detail "No resolved target for parameter '{id}'".

3. **Pre-validation pass (D-05):** For each parameter with a matched target:
   - If `target.Resolution == Missing` → `MissingTarget` with detail "No upstream source connected to DESIGN STATE input '{id}'"
   - If `target.Resolution == Ambiguous` → `AmbiguousTarget` with detail "Multiple upstream sources share ParameterId '{id}'"
   - If `target.TargetType` is null or doesn't match `parameter.Type` → `TypeMismatch` with detail "Expected {parameter.Type} but target is {target.TargetType}"
   - If target has domain bounds (`DomainMin`/`DomainMax` both non-null) AND the parameter's numeric value lies outside [min, max]:
     - For Number: check `parameter.NumberValue` against bounds
     - For Integer: check `parameter.IntegerValue` (cast to double) against bounds
     - Boolean: skip range check (no domain bounds concept)
     → `OutOfRange` with detail "Value {value} outside domain [{min}, {max}]"
   - Otherwise: mark as passing (tentatively `Applied`)

4. **Atomic decision (D-06):** If ANY parameter has a blocking status (MissingTarget, TypeMismatch, AmbiguousTarget, OutOfRange), flip ALL passing parameters to `WouldApply`. Set `Aborted = true`, `Applied = false`.

5. **Success path:** If all parameters pass pre-validation, set each to `Applied`. Set `Applied = true`, `Aborted = false`.

Namespace: `DG.Core.Services`. No GH references. Follow the same sealed-class, stateless style as `ValidationRunPersistenceService`.
