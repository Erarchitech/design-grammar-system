## Overview

v4.0 introduces automatic alignment of the DG Ontograph to the W3C Building Topology Ontology (BOT). When users submit rules containing spatial concepts (building, storey, space, site, zone, element), the pipeline detects the match via an in-memory vocabulary map and creates BOT anchor nodes on-demand in Neo4j, linked to the project's `ex:` OntoGraph nodes via `ALIGNED_TO` edges. Concepts with no BOT equivalent continue to create custom `ex:` entities as before.
