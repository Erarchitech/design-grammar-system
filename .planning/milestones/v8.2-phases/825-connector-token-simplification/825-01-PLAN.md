# Plan 825-01 — Backend: project-bound credentials + connection bundle

## Task 1 — Credential model gains `project` (connectors.py)

- `CredentialCreatePayload`: add `project: str | None = None`.
- `create_credential(connector_id, label=None, project=None)`: persist
  `"project": project or "default-project"` on the record.
- `HeartbeatResponse`: add `project: str` and `neo4j: Neo4jBundle` fields; add a
  `Neo4jBundle(BaseModel)` with `uri, user, password, database`.
- `record_heartbeat` unchanged (still returns the record; the route reads `project`).
- Backward tolerance: existing records without `project` read as `"default-project"`.

## Task 2 — Heartbeat returns the bundle (app.py)

- Add env reads near the existing NEO4J_* block:
  `NEO4J_PUBLIC_URI = os.getenv("NEO4J_PUBLIC_URI", "bolt://localhost:7687")`,
  `NEO4J_DATABASE = os.getenv("NEO4J_DATABASE", "neo4j")`.
- `connector_heartbeat`: build and return
  `HeartbeatResponse(connector_id, status, project=record.get("project","default-project"),
  neo4j=Neo4jBundle(uri=NEO4J_PUBLIC_URI, user=NEO4J_USER, password=NEO4J_PASSWORD,
  database=NEO4J_DATABASE))`.
- `create_connector_credential`: pass `payload.project` through.

## Task 3 — UI create-credential project field

- `connectorsApi.js createCredential(connectorId, label, project)`: include `project` in
  the POST body when set.
- `ConnectorsScreen.jsx CredentialSection`: add a small "Project" text input next to the
  Label input in the create form; thread a `projectInputs`/`onSetProjectInput` state pair
  through the same wiring as `labelInputs` (state lives in `ConnectorsScreen`). Default the
  value to the active project (`localStorage.dgv2_project` if present, else empty →
  server defaults to `default-project`).

## Task 4 — Tests (data-service/tests/test_connectors.py)

- Create-with-project persists `project` and the heartbeat echoes it.
- Heartbeat 200 body includes `neo4j.uri == bolt://localhost:7687` (host-facing, NOT
  `neo4j:7687`), plus user/password/database.
- Create without project → heartbeat `project == "default-project"`.
- Existing 401 unknown/revoked/malformed-auth tests still pass unchanged.
