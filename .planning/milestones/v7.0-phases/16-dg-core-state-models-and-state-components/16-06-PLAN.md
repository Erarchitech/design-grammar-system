<objective>
Create the PROPERTY STATE and DESIGN STATE composition Grasshopper components. PROPERTY STATE composes Rule+DataProperty+PropValue into a PropState with index-mismatch guard. DESIGN STATE composes independent ObjState/ParamState/PropState lists into a single aggregate DesignState.

Purpose: PROPERTY STATE enables architects to capture calculated property values from rule evaluation as a composable state fragment. DESIGN STATE aggregates all three state types into the single DesignState that VALIDATOR (Phase 18) consumes for validation run publishing.

Output: PropertyStateComponent.cs, DesignStateCompositionComponent.cs, updated DgIcons.cs, icon assets
</objective>
