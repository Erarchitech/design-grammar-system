# Plan 23-01: Graph Viewer — Live Neo4j Datascape

> Documented post facto per milestone directive.

## Objective

The datascape is real: the V2 canvas renders the live project-scoped Neo4j metagraph as the mockup's orbital ring layers (translucent ink arcs, achromatic dots, idle drift, divergence field), with node selection (red halo + inclined divergence callout + details panel), the persistent prompt bar wired to the rules-ingest and graph-query n8n webhooks with an extended session panel, and node search with a results popover that centres the canvas.

## Approach

- **`graphApi.js`** — Neo4j HTTP tx endpoint via nginx proxy (Basic auth per `config.template.js`), n8n webhooks with the legacy async contract (`accepted`/`executionId` → poll `data-service /execution-result/...`), project list query.
- **`buildRings.js`** — live graph → orbital model: one ring layer per `graph` property value (OntoGraph/Metagraph/KnowledgeGraph/SpecGraph/ValidGraph…), three orbits per layer bucketed by node label, display captions per the legacy label map; same-layer relationships → ring edges (typed), cross-layer → labelled cross edges.
- **`graphEngine.js`** — mockup canvas logic generalised from 5 fixed layers to N: camera pan/zoom-about-cursor, per-orbit drag rotation with inertia, click/double-click selection with fan alignment + fit-to-selection, hover, right-click search popover trigger, divergence field (LUT-based), spire callout, minimap with viewport rect + drag.
- **`GraphScreen.jsx`** — layers panel, orbit legend, hover/detail panels (engine-positioned), search popover with clickable results, property-scoped filter bar, prompt console (Ingest/Query tabs) with real step progress and session turns.

## Verification

Live against the Docker stack (all 14 containers): 6 ring layers built from 1317 real nodes; selection of `R_BUILDING_MAX_HEIGHT_75_V` (URBAN_BLOCK_V7) shows chip/properties/edges; search "height" → 17 matches → jump; UI query answered by the LLM in 4.7s in the session panel; UI ingest created `R_BLOCK_MIN_FOOTPRINT_PCT_18_V` with graph reload. Project scoping re-checked per ring counts.
