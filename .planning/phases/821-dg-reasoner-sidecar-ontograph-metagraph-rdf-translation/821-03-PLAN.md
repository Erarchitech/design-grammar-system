    `strip_hermit_unsupported(union)` and keep the returned `stripped_builtin_rules` count; compute

    axiom_counts, duration_ms, stripped_builtin_rules}`. On timeout return a clean 504-style dict
    `{consistent: null, error: "timeout", duration_ms, stripped_builtin_rules, timeout_seconds}`.

    real pySHACL `validate(data_graph, shacl_graph=<empty rdflib.Graph>, inference="none")`; return
