## Key Design Points

- `SharedProperty` MERGEd on `(dgId, propertyName, project)` — any bound platform sees the same value
- Each write records provenance: `platform`, `connector`, `writtenAt` timestamp
- `SharedPropertyWriteRequest` validates platform against the known PLATFORMS enum
- `read_shared_property` raises `DgIdentityError(DGID_NOT_FOUND)` for missing property (structured What+Where+How-to-fix)
- `list_shared_properties` returns all properties on a dgId; GET route switches on `?property_name=` query param
- Last-write-wins conflict policy (documented in spec/DG-ID.md)
