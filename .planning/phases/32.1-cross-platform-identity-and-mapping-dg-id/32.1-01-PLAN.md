<objective>
Create the GH-free deterministic DG ID minting core in DG.Core: a `DgId` value type and a `DgIdMintingService` that derives a durable, platform-neutral identity from a Computgraph entity's `(project, definitionId, cgId)`. This is the load-bearing primitive for the whole cross-platform identity spine — every downstream artifact (Computgraph entity wiring in Plan 05, the data-service registry API in Plan 03/04, the spec in Plan 02) keys on the string this service produces.

Purpose: DGID-01 — a platform-neutral `dgId`, deterministic across re-extractions and unique within a project, that never requires a persistence round-trip to "remember" itself. Determinism is what makes re-extraction stability free.
Output: 3 files — the value type, the minting service, and a unit-test suite proving stability, cross-project isolation, rename re-mint, and format.
</objective>
