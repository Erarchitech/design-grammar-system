## Golden Vector (cross-language anchor)

| Field | Value |
|-------|-------|
| project | `p1` |
| definitionId | `frame.gh` |
| cgId | `cg:1:proc:11_Proc` |
| hash input | `p1\|frame.gh\|cg:1:proc:11_Proc` |
| **expected dgId** | **`dg:BC8E62EE137E2B56`** |

Plan 03's Python `compute_dg_id` MUST reproduce this exact input→output pair. Contract: pipe-join the three fields in this order, SHA-256 over UTF-8 bytes, uppercase hex, take the first 16 characters, prefix with `dg:`.
