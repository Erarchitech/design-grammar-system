# Plan 21-01: V2 App Scaffold, Tokens, Fonts, Primitives, Specimen

> Documented post facto per milestone directive (autonomous execution, planning artifacts written at completion).

## Objective

Stand up the new V2 UI app (`ui-v2/`, per D-10 discretion) as a Vite + React (JSX, JavaScript) project carrying the complete design-system foundation: the 6-file token stylesheet split, self-hosted Geist/Geist Mono/Oswald, and all ~23 component recipes from `design/v2/_ds/`, proven by a dev-only specimen page.

## Context inputs

- `.planning/phases/21-design-system-foundation/21-CONTEXT.md` (decisions D-01..D-11)
- `design/v2/_ds/.../_ds_manifest.json` + `_ds_bundle.js` — the component recipes exist only as Babel-compiled `React.createElement` code inside the bundle (no `components/*.jsx` on disk); recipes were reconstructed as clean JSX from the compiled forms
- `design/v2/_ds/.../tokens/*.css` — copied byte-for-byte, then adapted (see decisions)

## Tasks

1. **Scaffold ui-v2 Vite+React app with token stylesheets and self-hosted fonts**
   - `ui-v2/` sibling of `graph-viewer/` (D-10); React 18.3 + Vite 5.4 mirroring model-viewer versions
   - Dev proxy `/neo4j` `/n8n` `/data-service` → `http://localhost:8080` (D-11)
   - Token CSS 6-file split preserved (D-05); fonts.css rewritten with local `@font-face` over woff2 binaries vendored from @fontsource latin subsets (D-06): Geist 400/500/600, Geist Mono 400/500, Oswald 400/500/600
2. **Port the 23-component primitive library** to `src/components/{forms,display,surfaces}/` (D-04, D-08) with barrel `index.js`
3. **Dev-only specimen page** (D-09) at `/#specimen` (hash check, no router dependency), organised Colors / Type / Foundations / Components like the _ds guideline cards

## Verification (success criteria → checks)

1. Token stylesheet loads; three-tone surface stack; Signal Red only chromatic → specimen swatches + computed body bg `rgb(245,245,245)`
2. Geist/Geist Mono/Oswald in spec roles → `document.fonts` reports all 8 local faces loaded
3. Primitives per recipes → computed styles: button radius 18px @ 36px height, frost `rgba(255,255,255,0.78)` + `blur(14px)` + hairline, annotation Oswald uppercase +1.2px
