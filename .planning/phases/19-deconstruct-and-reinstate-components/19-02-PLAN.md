<objective>
Create two passive deconstruction components — DESIGN STATE DECONSTRUCT splits a DesignState into three typed lists, OBJECT DECONSTRUCT splits an ObjState into its property values.

Purpose: Architects can disassemble stored DesignStates back into individual ObjState/ParamState/PropState entities, and further split each ObjState into the Object(Ref)/Geometry/Label — enabling inspection, re-wiring, or selective processing.

Output: Two GH_Component files following the pure synchronous passthrough pattern (per GraphDeconstructComponent precedent from Phase 17), with matching test files in DG.Tests.
</objective>

    - description: "Split a DesignState into ObjState, ParamState, and PropState lists. Pure synchronous passthrough -- no refresh required."
    - category: DgComponentCategory.Category
    - subcategory: DgComponentCategory.Subcategory

    GUID: `6D7B8C9D-0E1F-4A2B-8C3D-4E5F6A7B8C9D`

    INPUTS:
    - Index 0: GenericParameter "DesignState" / "DesignState" / item / "DG.DesignState -- aggregate state from DESIGN STATE or VALIDATION GRAPH."

    - Index 0: GenericParameter "ObjState" / "ObjState" / list / "List of ObjState objects. From DesignState.ObjStates."
    - Index 1: GenericParameter "ParamState" / "ParamState" / list / "List of ParamState objects. From DesignState.ParamStates."
    - Index 2: GenericParameter "PropState" / "PropState" / list / "List of PropState objects. From DesignState.PropStates."

    SolveInstance:
    1. da.GetData(0, ref input) -- if false: Warning + SetEmptyOutputs + return
    2. TryDesignState(input) -- if null: Warning + SetEmptyOutputs + return
    3. Cache to _latest* fields
    4. da.SetDataList(0, designState.ObjStates) -- output Core model types directly
    5. da.SetDataList(1, designState.ParamStates)
    6. da.SetDataList(2, designState.PropStates)
    7. Message = $"{designState.ObjStates.Count}O/{designState.ParamStates.Count}P/{designState.PropStates.Count}Pr"

    SetEmptyOutputs:
    - Sets all 3 outputs to empty lists (new List<ObjState>(), new List<ParamState>(), new List<PropState>())
    - Adds Remark: status string
    - Clears cached _latest* fields

    Follow the D-07 contract: Warning for null/missing input, no warning for empty bags.
    The GRASSHOPPER_SDK guard and #else stub follow the same pattern as GraphDeconstructComponent.
