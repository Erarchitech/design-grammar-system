<objective>
Create the Grasshopper infrastructure layer: PublicType wrappers for the four new state models, GhCastingHelpers unwrappers, and ErrorMessageTemplates index-mismatch messages. This plan provides the GH data-flow contract that downstream component plans (05, 06) will use.

Purpose: GH components cannot wire raw Core types as output data — they need a public wrapper (GH data type) defined in the DG.Grasshopper assembly. GhCastingHelpers provides the dual-path unwrapping (Core type directly or via public wrapper). ErrorMessageTemplates provides standardized user-facing error messages for parallel-list index mismatches.

Output: Updated PublicTypes.cs, GhCastingHelpers.cs, ErrorMessageTemplates.cs
</objective>
